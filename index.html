Personal Wedding Inspiration MCP Tool

A personal, non-commercial MCP server integration used to connect an authorized Pinterest account through OAuth 2.0 and organize visual inspiration for a wedding website project.

Purpose

This tool is used by its owner to search for and organize wedding design inspiration, including Pins and boards related to wedding website visuals, color palettes, layout references, typography, floral styles, and photography moodboards.

The integration is intended only for personal use while creating a wedding website. It is not a public consumer product, advertising tool, analytics service, or data resale service.

Pinterest API Use

The tool uses Pinterest OAuth 2.0. It does not collect Pinterest usernames, passwords, session cookies, or login credentials. API access is used only after the authorized user grants permission through Pinterest's OAuth flow.

OAuth tokens are stored locally in the owner's development environment and are not sold, shared, transferred, or used for third-party advertising profiling.
