Privacy Policy

Last updated: June 18, 2026

This Privacy Policy describes how the Personal Wedding Inspiration MCP Tool handles information when used to connect an authorized Pinterest account for a personal wedding website project.

Purpose of the Tool

This is a personal, non-commercial tool used by its owner to search for and organize wedding design inspiration from Pinterest through the Pinterest API.

Information Accessed

Depending on the permissions approved through Pinterest OAuth, the tool may access Pinterest boards, Pins, search results, and related account content made available by the Pinterest API.

OAuth and Credentials

The tool uses Pinterest OAuth 2.0. It does not collect Pinterest usernames, passwords, session cookies, or login credentials.

OAuth tokens may be stored locally in the owner's development environment so the tool can make authorized API requests. Tokens are not sold, shared, transferred, or used for third-party advertising profiling.

How Information Is Used

Pinterest API data is used only to help the owner find, review, and organize visual references for a personal wedding website, including style references, layout ideas, color palettes, typography, floral ideas, and photography moodboards.

Sharing

Information accessed through the Pinterest API is not sold, rented, shared with third parties, or used for any public user-facing service.

Data Deletion

Because this is a personal local tool, stored OAuth tokens and any locally saved Pinterest API data can be deleted from the owner's development environment at any time.

Contact

For privacy questions or deletion requests, contact: crowdwide@tmail.ae.
